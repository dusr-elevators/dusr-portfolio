# Design Page CTA Button Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a floating "Design Your Elevator" CTA button to the home page that links to `/design`, with a Django-admin checkbox to hide/show it.

**Architecture:** A new `DesignCTASettings` singleton model in the `design` Django app (admin-editable via a one-click `list_editable` checkbox), exposed through a read-only DRF endpoint `GET /api/design/cta-settings/`. The Next.js home route fetches that flag server-side and passes it down to a new `DesignCTAButton` client component rendered inside `Home.tsx`.

**Tech Stack:** Django + Django REST Framework (backend/design app), Next.js App Router + React + `motion/react` + `lucide-react` + Tailwind (frontend).

## Global Constraints

- Backend dev settings module: `goldenMeatPortfolio.settings.dev` (default in `manage.py`); run all backend commands from the `backend/` directory with `.venv` activated.
- Frontend commands run from the `frontend/` directory (`npm run typecheck`, `npm run lint`).
- No JS test runner exists in this repo — per [[frontend-tests-are-typechecked-docs]], frontend correctness is verified by `tsc --noEmit` plus manual browser verification, not automated test files.
- DRF defaults to `AllowAny`/no authentication (`backend/goldenMeatPortfolio/settings/base.py:190-193`) — the new endpoint needs no auth handling.
- Button copy: "Design Your Elevator" (en) / "صمم مصعدك" (ar). Links to `/design` (ar) or `/en/design` (en).
- Button position: fixed bottom-right for `lang === 'en'`, fixed bottom-left for `lang === 'ar'` (mirrors `Hero.tsx`'s existing `lang === 'ar' ? 'left-6 md:left-16' : 'right-6 md:right-16'` convention).
- Button z-index: `z-[60]` (above `Header.tsx`'s `z-50`).
- Fetch failure behavior: default to `showDesignCTA = false` (fail closed).

```tsx
/**
 * @license
 * SPDX-License-Identifier: Apache-2.0
 */

import type { Metadata } from 'next';
import { notFound } from 'next/navigation';
import { isLang } from '@/lib/lang';
import { homeMetadata } from '@/lib/seo';
import Home from '@/components/Home';

const apiBase = process.env.API_INTERNAL_URL || 'http://localhost:8000';

async function fetchShowDesignCTA(): Promise<boolean> {
  try {
    const res = await fetch(`${apiBase}/api/design/cta-settings/`, { cache: 'no-store' });
    if (!res.ok) return false;
    const data = await res.json();
    return Boolean(data.is_visible);
  } catch {
    return false;
  }
}

export async function generateMetadata({
  params,
}: {
  params: Promise<{ lang: string }>;
}): Promise<Metadata> {
  const { lang } = await params;
  if (!isLang(lang)) return {};
  return homeMetadata(lang);
}

export default async function Page({ params }: { params: Promise<{ lang: string }> }) {
  const { lang } = await params;
  if (!isLang(lang)) notFound();
  const showDesignCTA = await fetchShowDesignCTA();
  return <Home lang={lang} showDesignCTA={showDesignCTA} />;
}
```
